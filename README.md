## Overview

Quite often we want to share our visualizations and graphics as part of a presentation. In this lab we are going to learn the basics of building a Presentation with Quarto. Previously we used Quarto to create our homework documents because it allows use to interlace prose, code, and computational outputs seamlessly in a flexible, organized, neat, and dare I say, stylish manner. The same reasons should motivate us to use Quarto for building presentations.

## Instructions

After creating your own GitHub repository and connecting it to your RStudio, you should work through the exercises provided in `L13_presentations.html`. A template is also provided (rename as `Lastname_Firstname_L13.qmd`).

You will only need to submit the rendered html file which **must contain a link to your github repo**. Including the github repo link provides all the support materials needed to reproduce your work, an essential feature of quality scientific work, especially data science/analysis work. See canvas for submission details.
